# Racket-AprendizajeSupervisado
Desarrollo de algoritmos de aprendizaje supervisado

El presente proyecto contiene el desarrollo de los ejercicios de "Métodos de Aprendizaje en Inteligencia Artificial"
de Félix Hernandez del Olmo (Dpto. Inteligencia Artificial, UNED) 2006-2007.

El lenguaje en el que se desarrolla el presente proyecto es Racket, R5RS, una distribución de LISP, en el software DrRaket, 
mismo que puede ser descargado desde el siguiente url: https://download.racket-lang.org/
